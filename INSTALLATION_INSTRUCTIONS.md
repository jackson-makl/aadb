### ‚ùì Troubleshooting

#### "App is damaged and can't be opened"
If you see this error, open Terminal and run:
```bash
xattr -cr /Applications/AADB_Risk_Predictor.app
```
Then try opening again.

#### App won't start
1. Make sure you're on macOS 10.13 or newer
2. Try restarting your Mac
3. Delete the app and re-download/copy it

#### Multiple windows opening
This shouldn't happen anymore, but if it does:
1. Close all windows
2. Wait 10 seconds
3. Open the app again
